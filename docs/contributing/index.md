# Contributing

Thank you for your interest in contributing to Machineconfig!

---

## Ways to Contribute

- **Report bugs** - Found a bug? [Open an issue](https://github.com/thisismygitrepo/machineconfig/issues/new)
- **Suggest features** - Have an idea? We'd love to hear it
- **Fix bugs** - Browse [open issues](https://github.com/thisismygitrepo/machineconfig/issues) and submit PRs
- **Improve documentation** - Help make our docs better
- **Share feedback** - Let us know how we can improve

---

## Getting Started

1. Fork the repository
2. Clone your fork
3. Set up development environment
4. Make your changes
5. Submit a pull request

See [Development Setup](development.md) for detailed instructions.

## Code of Conduct

Please read our [Code of Conduct](https://github.com/thisismygitrepo/machineconfig/blob/main/CODE_OF_CONDUCT.md) before contributing.
